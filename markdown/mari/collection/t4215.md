# Fujitsu Lifebook T4215
---
The Fujitsu Lifebook T4215 is a Socket M laptop made by Fujitsu-Seimens, sometime in 2006. It uses Merom CPUs, and has GMA 950 Integrated Graphics.

## Specifications
| Component | Spec |
|-----------|------|
| CPU      | Intel Core 2 Duo T7400 |
| RAM | 4GB DDR2-667 Memory (3.25 GB Usable) |
| Storage | 180GB Intel Pro 1500 SSD |
| Display | 12.1" 1024x768 LCD |
| Graphics | Intel GMA 950 |
| Wi-Fi | Atheros Chipset |
| Optical Drive | None |
| ODM | Unknown |

## History
I bought it off of ebay for $50. I wanted some form of Tablet laptop for many years and now I finally got one. I bought it and instantally fell in love with it. I upgraded the CPU from the T5500 to the T7400, RAM from 1.5 GB to 4 GB (with only 3.25 GB of it usable due to chipset limitations), and Installed an SSD.
